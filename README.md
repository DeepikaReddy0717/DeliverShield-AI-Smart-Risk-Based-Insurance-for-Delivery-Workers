## Inspiration

Delivery workers face unpredictable risks due to extreme weather and pollution. Traditional insurance does not adapt to real-time conditions. This inspired us to build **DeliverShield AI**, a smarter and dynamic insurance system.

---

## What it does

**DeliverShield AI** analyzes:
- Temperature 🌡
- Weather 🌧
- Air Quality 🌫

It calculates a **risk score** and classifies workers into:
- Low Risk
- Medium Risk
- High Risk

---

## How we built it

We used:

- **Django** for backend
- **SQLite** for database
- **HTML, CSS, Bootstrap** for UI
- **OpenWeatherMap API** for real-time data

Example logic:
-**Risk = Heat + Rain + Pollution**
